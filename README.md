# Amazon_Sales_dashboard
1. Objective
Analyze Amazon sales data to identify trends, customer behavior, and business insights.


2. Dataset Overview
The dataset includes Order ID, Date, Status, Fulfillment type, Category, Size, Quantity, Amount,
and Shipping details.

3. Key Metrics
Total Sales: 78.50M
Total Orders: 120K
Total Quantity: 116K


4. Key Insights
- T-shirts are the highest revenue-generating category.
- Amazon fulfillment performs better than Merchant.
- Bangalore, Hyderabad, and Mumbai generate the highest sales.
- Sizes M and L are most preferred by customers.
- Most orders are successfully shipped with few cancellations.


5. Recommendations
- Increase inventory for high-demand products.
- Focus marketing efforts in top-performing cities.
- Improve merchant fulfillment efficiency.
- Optimize stock planning for popular sizes.
- Reduce cancellations through better logistics.


6. Conclusion
This analysis helps businesses improve decision-making, optimize inventory, and enhance
customer satisfactio
